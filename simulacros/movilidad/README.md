**Contexto:** Una empresa de movilidad necesita un componente de software para administrar su flota de vehículos, calcular costos de alquiler y generar reportes operativos. El sistema debe soportar distintos tipos de vehículos, cada uno con sus propias reglas de tarificación y mantenimiento.

**Requerimientos:**

1. El vehículo podrá ser: Auto, Camioneta, Motocicleta, Furgón.

2. Se podrá calcular el costo total de alquiler según los días contratados y la tarifa base específica de cada tipo.

3. Cada vehículo tendrá una patente, marca, modelo y un estado (DISPONIBLE o EN_MANTENIMIENTO).

4. Se deberá poder listar los vehículos DISPONIBLES ordenados por costo de alquiler diario (de menor a mayor).

5. Se deberá poder identificar el vehículo que requiere el menor costo de mantenimiento mensual.

Se PIDE:

1. Diagrama de Clases Alto y/o bajo nivel que modele los requerimientos propuestos.

2. Diagrama de Secuencia que ejemplifique un escenario completo de registro y consulta.
